Focused on

App Performance

One of the main reasons users don't visit some websites more than once is represented by the page loading time.

That's why I emphasize the importance of app performance in terms of page speed and response time in every project I'm involved in.

Clean Code

Keeping a well-organized clean repository should be every developer's goal. Code should be self-explanatory, easily maintainable, and well-documented.

This is what I also try to do whenever I work on something — old or new — and I try to "sell" the same principle to others.

Adaptive / Responsive Design

Mobile-first approaches should be the industry standard nowadays.

By adopting this idea when developing an app you achieve the best user experience with one codebase, no matter the device it runs on.
